# Game-Ying-online Termux/window
## วิธีรันบน Termux
```
pkg install nodejs -y
```
```
git clone https://github.com/ninjamadeena/Game-Ying-online
```
```
cd Game-Ying-online
```
```
npm install -D nodemon
```
รันใช้
```
node server.js
```
